# モダンToDoアプリ 追加機能アイデア

現在のCloudflare Workers (Hono) + D1 によるSPAの高速な構成を活かして、将来的に追加するとさらに便利になる機能のアイデア一覧です。

## 1. ユーザー認証機能（ログイン機能）
現在はURLを知っていれば誰でも同じタスクリストにアクセスできてしまうため、GoogleアカウントやGitHubでのログイン機能（OAuthなど）を追加する。
これにより、複数のユーザーがそれぞれ自分専用のToDoリストを管理できるようになります。

## 2. [実装済] PWA（プログレッシブウェブアプリ）対応
PCブラウザだけでなく、スマホ環境でも快適に利用できるように `manifest.json` や Service Worker を追加しました。
スマホのホーム画面にアイコンを追加して、通常のネイティブアプリのように全画面で起動でき、オフライン対応機能なども実装されています。

## 3. [実装済] ダークモード / ライトモードの切り替え切替
現在のグラスモーフィズムを取り入れたダークテーマに加え、昼間や明るい場所で使いやすい「ライトテーマ」を用意しました。
画面右上にテーマ切り替えのトグルボタンを設置し、ユーザーの好みに応じてデザインを変更できるようになっています。

## 4. タスクの詳細メモ・サブタスク機能
タスクの中にさらに細かく分解した「サブタスク（小さなチェックリスト）」を追加できるようにする。
また、タスクをクリックした際にモーダルを開き、詳細な手順やMarkdown形式でのメモを記述できる機能です。

## 5. PC向けのキーボードショートカット
マウスを使わずにキーボードだけでサクサク操作できるようにするための機能。
- 例：`n` を押すとタスク追加欄にフォーカス
- 例：上下矢印キー `↑` `↓` でタスクを選択し、`Space` キーで完了状態を切り替え

## 6. メトリクスやグラフの表示（統計機能）
「今週は〇個のタスクを完了しました」といったモチベーションアップに繋がる情報を表示する機能。
ホーム画面の隅に完了率の簡単な円グラフや、過去1週間のタスク消化数の棒グラフなどを小さく表示させます。
